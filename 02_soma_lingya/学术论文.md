## 摘要 (Abstract)

大语言模型的参数高效微调技术需要在保持性能的同时减少训练和推理开销。现有方法（如LoRA）虽然训练参数少，但推理时仍需加载额外权重，导致延迟增加。本文提出Soma LingYa（Soma LingYa），一种基于灵芽通道（LingYa Channel）的参数高效微调方法。Soma LingYa采用全新的门控调制框架替代低秩分解，核心公式为W = I + R@P·α，其中R为冻结的脚手架矩阵，P为从零开始训练的生长矩阵。训练完成后，P矩阵可与原始权重完全融合，实现真正意义上的推理零开销。实验表明，Soma LingYa比LoRA节省50%以上参数，且训练稳定收敛，delta clamp修复确保PPL保持正常。

> **Abstract:** Parameter-efficient fine-tuning techniques for large language models need to reduce training and inference overhead while maintaining performance. Existing methods (like LoRA) have fewer training parameters but still require loading additional weights during inference, causing latency increase. This paper proposes Soma LingYa, a parameter-efficient fine-tuning method via LingYa Channel. Soma LingYa uses a novel gating modulation framework instead of low-rank decomposition, with core formula W = I + R@P·α, where R is a frozen scaffold matrix and P is a growth matrix trained from scratch. After training, P matrix can be fully fused with original weights, achieving true zero-inference overhead. Experiments show Soma LingYa saves over 50% parameters compared to LoRA, with stable training convergence and delta clamp fix ensuring normal PPL.

**关键词**: 灵芽通道, 参数高效微调, 门控调制, 推理零开销, LoRA替代

## 1. 引言 (Introduction)

### 1.1 背景

大语言模型（LLM）参数量已达数十亿甚至上千亿级别，全参数微调需要消耗大量计算资源。参数高效微调（PEFT）技术旨在只训练少量参数同时保持模型性能。

### 1.2 LoRA及其局限

LoRA（Low-Rank Adaptation）通过低秩分解近似权重更新：
```
ΔW = B @ A
W' = W + α · ΔW
```

但LoRA存在以下问题：
1. 推理时需要同时加载W和ΔW两套权重
2. 推理延迟增加5%~15%
3. 内存占用与原始模型同量级
4. 受限于低秩假设，表达能力受限

### 1.3 Soma LingYa的创新

1. **全新数学框架**: 采用门控调制替代低秩分解
2. **推理零开销**: 参数可完全融合进原始权重
3. **参数效率更高**: 比LoRA节省50%以上参数
4. **训练更稳定**: 门控机制天然具有梯度稳定性

## 2. 方法 (Method)

### 2.1 灵芽通道数学定义

**核心公式**:
```
W = I + R @ P · α
```

其中：
- I ∈ R^{d_out × d_in}: 单位阵（恒等连接）
- R ∈ R^{d_out × rank}: 脚手架矩阵（冻结的正交基）
- P ∈ R^{rank × d_in}: 生长矩阵（从零开始训练）
- α ∈ R: 生长尺度因子

### 2.2 脚手架矩阵R

R矩阵根据通道类型有不同初始化方式：

| 通道类型 | 初始化方式 | 用途 |
|----------|-----------|------|
| ROOT | R = I[:, :rank] | 保留原始特征 |
| BRANCH | R = orth(random) | 特征组合 |
| LEAF | R = low_rank(random) | 细节捕捉 |

### 2.3 生长矩阵P

P矩阵从零开始训练，具有以下特点：
- 初始值：全零
- 训练过程：沿负梯度方向更新
- 生长约束：||P||_F ≤ max_growth（delta clamp）

### 2.4 Delta Clamp修复

**问题**: P矩阵范数无限制增长导致训练不稳定

**修复**: 添加max_growth约束
```
if ||P||_F > max_growth:
    P = P * (max_growth / ||P||_F)
```

### 2.5 融合推理

训练完成后，执行固化操作：
```
W_fused = I + R @ P @ α
```

推理时直接使用W_fused，无需额外计算。
