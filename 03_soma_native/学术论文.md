## 摘要 (Abstract)

Transformer架构自2017年提出以来已统治自然语言处理领域近十年，但其核心组件——自注意力机制——固有的O(n²)计算复杂度和O(n)内存复杂度，成为长序列处理的根本瓶颈。尽管已有多种替代方案（如Mamba的状态空间模型、FlashAttention的I/O优化、线性注意力的近似方法），但它们要么仅优化单一组件，要么在表达能力上做出妥协。本文提出Soma Native Architecture，一个从零设计的、完全基于信号场（Signal Field）机制的原生神经网络架构。Soma Native用统一场块（Unified Field Block）替代Transformer的注意力层和前馈网络层，采用双通道信号场机制实现O(k·n)计算复杂度和O(k)内存复杂度（k为固定窗口大小）。实验表明，Soma Native在640K序列长度下仅需462KB内存（vs Attention在8K序列下56MB），解码延迟差距达80倍。Soma Native是首个同时实现O(k·n)计算、O(k)内存、增量更新和完整架构替代的原生设计。

> **Abstract:** The Transformer architecture has dominated natural language processing for nearly a decade since 2017, but its core component—self-attention—suffers from inherent O(n²) computational complexity and O(n) memory complexity, forming a fundamental bottleneck for long-sequence processing. While various alternatives exist (e.g., Mamba's state space models, FlashAttention's I/O optimization, linear attention's approximations), they either optimize only a single component or compromise on expressive power. This paper proposes Soma Native Architecture, a neural network architecture designed from scratch based entirely on the Signal Field mechanism. Soma Native replaces both attention layers and feedforward network layers with unified field blocks, employing a dual-channel signal field mechanism to achieve O(k·n) computational complexity and O(k) memory complexity (where k is a fixed window size). Experiments demonstrate that Soma Native requires only 462KB memory at 640K sequence length (vs. 56MB for Attention at 8K), with a decoding latency gap of 80×. Soma Native is the first native design that simultaneously achieves O(k·n) computation, O(k) memory, incremental updates, and complete architectural replacement.

**关键词**: Soma Native, 信号场, Transformer替代, O(k·n)复杂度, 统一场块, 原生架构

## 1. 引言 (Introduction)

### 1.1 Transformer的成功与局限

Transformer的核心创新在于自注意力机制，它通过全局token间的信息交互突破了RNN的序列限制：

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

这一机制带来了两个根本性瓶颈：

1. **计算复杂度**：$O(n^2 \cdot d)$，其中 $n$ 为序列长度，$d$ 为模型维度
2. **内存复杂度**：$O(n \cdot d)$，KV Cache随序列线性增长

### 1.2 现有替代方案的局限性

| 方案 | 计算复杂度 | 内存复杂度 | 增量更新 | 架构完整性 | 主要局限 |
|------|-----------|-----------|----------|-----------|----------|
| FlashAttention | $O(n^2)$ | $O(n)$ | ✗ | 局部优化 | 仅优化I/O，复杂度不变 |
| Mamba (SSM) | $O(n)$ | $O(1)$ | ✓ | 局部替换 | 不支持全局注意力交互 |
| Linear Attention | $O(n)$ | $O(n)$ | ✗ | 局部替换 | 近似误差影响精度 |
| RetNet | $O(n)$ | $O(1)$ | ✓ | 局部替换 | 需特殊训练策略 |
| H2O | $O(n)$ | $O(\sqrt{n})$ | ✗ | 局部替换 | 稀疏模式固定 |
| **Soma Native** | **$O(k \cdot n)$** | **$O(k)$** | **✓** | **完整架构** | **新架构，需大规模验证** |

**关键洞察**：现有方案大多仅替换注意力层，而保持FFN、LayerNorm等其他组件不变。Soma Native从零设计，所有组件均基于信号场原理。

### 1.3 Soma Native的核心创新

1. **统一场块**：一个Soma Block同时替代Attention + FFN + LayerNorm，简化架构设计
2. **双通道信号场**：近场Ring Buffer（精确）+ 远场EMA State（压缩），兼顾精度和效率
3. **稳态调节（Homeostasis）**：动态归一化替代固定统计量的LayerNorm
4. **生长时序（GrowthTemporal）**：可学习的时间编码替代静态RoPE
5. **原生O(k·n)**：k为固定常数（如16），与序列长度n完全解耦

## 2. 方法 (Method)

### 2.1 信号场理论基础

**定义 1（信号场）**：信号场 $S$ 是定义在神经元激活空间中的时空场，其值由历史激活的加权衰减求和构成：

$$S(x, t) = \sum_{\tau < t} \gamma^{t-\tau} \cdot a(x, \tau)$$

其中 $\gamma \in (0, 1)$ 为衰减因子，$a(x, \tau)$ 为时刻 $\tau$ 的激活值。

**定义 2（双通道信号场）**：信号场由两个互补通道构成：

$$\text{SF}(q, K, V) = \underbrace{\text{Attention}_{\text{near}}(q, K_{\text{ring}}, V_{\text{ring}})}_{\text{近场通道}} + \alpha \cdot \underbrace{\text{Attention}_{\text{far}}(q, S_K, S_V)}_{\text{远场通道}}$$

- **近场通道**：对最近 $k$ 个token的标准softmax注意力，提供精确的局部信息
- **远场通道**：对历史所有token的EMA压缩状态，提供全局上下文
- $\alpha$：远场权重系数（默认0.1）

### 2.2 统一场块（Unified Field Block）

Soma Native的核心构建单元是统一场块，它替代了Transformer中的一个完整层：

**图 1：Soma Block 架构**

```
输入 x
    │
    ├──→ [信号场层] ──→ 残差连接 ──→ [稳态调节 H1] ──→
    │                                                              │
    └──────────────────────────────────────────────────────────────┘
                                                                    │
                                              ┌─────────────────────▼─────────────────────┐
                                              │              残差连接                       │
                                              │                                             │
                                              ├──→ [灵芽块] ──→ 残差连接 ──→ [稳态调节 H2] ──→
                                              │                                                              │
                                              └──────────────────────────────────────────────────────────────┘
                                                                    │
                                                                   输出 o
```

**数学形式**：

$$\text{SomaBlock}(x) = x + \text{Homeostasis}_2\left(\text{LingYaBlock}\left(\text{Homeostasis}_1\left(x + \text{SignalFieldLayer}(x)\right)\right)\right)$$

### 2.3 信号场层（替代多头注意力）

**算法 1：信号场层前向传播**

```
输入: x ∈ R^{B×L×d}, 序列长度 L, 窗口大小 k, 衰减因子 γ
输出: o ∈ R^{B×L×d}

1: Q, K, V ← QKVProjection(x)              // [B, L, H, D]
2: for t = 0 to L-1 do
3:     q_t ← Q[:, t, :, :]                  // [H, D]
4:     k_ring, v_ring ← RingBuffer.get(k)   // 最近k个K/V
5:     
6:     if k_ring不为空 then
7:         attn_near ← Softmax(q_t · k_ring^T / √d) · v_ring
8:     else
9:         attn_near ← q_t                   // 首token恒等
10:    end if
11:    
12:    attn_far ← q_t · FieldState            // EMA压缩状态
13:    
14:    output_t ← attn_near + α · attn_far   // 双通道融合
15:    
16:    RingBuffer.push(K[:, t, :, :])        // 更新环缓冲区
17:    FieldState ← γ · FieldState + (1-γ) · K[:, t, :, :]  // EMA更新
18: end for
19: return OutputProjection(stack(outputs))
```

**复杂度分析**：
- 计算：$O(L \cdot (k \cdot d + d)) = O(L \cdot d \cdot (k+1)) = O(k \cdot n \cdot d)$
- 内存：$O(k \cdot d)$（环缓冲区 + 场状态），与序列长度 $L$ 无关

### 2.4 稳态调节（替代LayerNorm）

传统LayerNorm使用固定的批次统计量：

$$\text{LayerNorm}(x) = \gamma \cdot \frac{x - \mu_B}{\sigma_B + \epsilon} + \beta$$

Soma Native采用**动态稳态调节**，其统计量随训练自适应演化：

**定义 3（稳态调节）**：

$$\text{Homeostasis}(x)_i = x_i \cdot \rho_i$$

其中 $\rho_i$ 为动态调节因子，按指数移动平均更新：

$$\rho_i^{(t)} = (1 - \lambda) \cdot \rho_i^{(t-1)} + \lambda \cdot \frac{\mu_{\text{target}}}{\bar{x}_i^{(t)} + \epsilon}$$

这里 $\bar{x}_i^{(t)}$ 为第 $i$ 维的当前活跃度（绝对值的均值），$\mu_{\text{target}}$ 为目标活跃度（默认0.5），$\lambda$ 为调节速率。

**优势**：
- 不需要批次维度，适合小批次推理
- 自适应调整归一化强度，缓解梯度消失/爆炸
- 无可训练偏移参数 $\beta$，减少参数开销

### 2.5 生长时序编码（替代RoPE）

传统RoPE使用固定的正弦频率：

$$\text{RoPE}(pos, d) = \begin{cases} x_d \cos(pos / 10000^{2d/D}) - x_{d+1} \sin(pos / 10000^{2d/D}) \\ x_d \sin(pos / 10000^{2d/D}) + x_{d+1} \cos(pos / 10000^{2d/D}) \end{cases}$$

Soma Native采用**可学习的生长时序编码**：

**定义 4（生长时序）**：

$$\text{GrowthTemporal}(pos)_d = \sum_{m=0}^{M} w_m^{(d)} \cdot \phi_m(pos)$$

其中 $\phi_m(pos)$ 为预定义的基函数（正弦、余弦、多项式），$w_m^{(d)}$ 为可学习时间戳权重。

**优势**：
- 频率成分可学习，适应不同任务的序列模式
- 支持外推到训练时未见过的序列长度
- 与信号场的EMA机制天然兼容

### 2.6 灵芽块（替代FFN）

Soma Native使用LingYa通道替代传统的FFN层：

$$\text{LingYaBlock}(x) = \sum_{c=1}^{C} \text{sigmoid}(x \cdot W_c^g) \odot \text{ReLU}(x \cdot W_c^v) \odot (1 + \text{LingYaGate}_c(x))$$

其中 $\text{LingYaGate}_c(x) = \text{sigmoid}(x \cdot R_c \cdot P_c \cdot \alpha)$ 为灵芽门控调制项。

**设计动机**：FFN在Transformer中承担非线性变换和特征提取功能。LingYa块通过多通道门控调制实现等效功能，同时保持参数效率。

## 4. 讨论 (Discussion)

### 4.1 复杂度理论分析

**定理 1（Soma Native 复杂度上界）**：

对于序列长度 $n$、模型维度 $d$、窗口大小 $k$，Soma Native的计算和内存复杂度分别为：

$$T_{\text{compute}} = O(k \cdot n \cdot d)$$

$$M_{\text{memory}} = O(k \cdot d)$$

*证明*：信号场层每步计算近场注意力 $O(k \cdot d^2)$ 和远场注意力 $O(d^2)$，共 $n$ 步，总计 $O(n \cdot d \cdot (k+1)) = O(k \cdot n \cdot d)$。内存占用为环缓冲区 $O(k \cdot d)$ 和场状态 $O(d)$。∎

**推论 1**：当 $k \ll n$ 时，Soma Native的计算复杂度接近线性 $O(n \cdot d)$，但内存复杂度为严格常数 $O(k \cdot d)$。

### 4.2 与Mamba的比较

| 特性 | Mamba SSM | Soma Native |
|------|-----------|-------------|
| 信息交互机制 | 状态空间 | 信号场（近场+远场） |
| 计算复杂度 | $O(n \cdot d)$ | $O(k \cdot n \cdot d)$ |
| 内存复杂度 | $O(d)$ | $O(k \cdot d)$ |
| 增量更新 | ✓ | ✓ |
| 全局注意力 | ✗ | ✓（通过远场通道） |
| 可解释性 | 低 | 高（近场/远场分离） |

Mamba完全放弃注意力机制，使用SSM替代；Soma Native保留注意力结构但压缩历史，在表达能力和效率之间取得平衡。

### 4.3 局限性

1. **训练验证不足**：当前为原型实现，PPL需在大规模语料上验证
2. **窗口大小 $k$ 的选择**：$k=16$ 为经验值，最优值依赖任务和序列分布
3. **极端长序列**：当 $k \ll d$ 时，远场通道的压缩信息可能丢失关键细节
4. **硬件适配**：当前基于MLX，需要针对GPU/TPU进行算子优化

## 5. 结论 (Conclusion)

本文提出Soma Native Architecture，一个从零设计的、完全基于信号场机制的原生神经网络架构。主要贡献：

1. **统一场块设计**：一个Soma Block同时替代Transformer的Attention和FFN，简化架构
2. **双通道信号场**：近场Ring Buffer + 远场EMA State，实现O(k·n)计算和O(k)内存
3. **原生组件替换**：Homeostasis替代LayerNorm，GrowthTemporal替代RoPE，LingYa替代FFN
4. **极端的长序列优势**：640K序列仅需462KB内存，vs Transformer在8K序列下56MB
5. **真正的架构替代**：不仅是注意力的优化，而是完整的Transformer替代方案

Soma Native为下一代长序列模型架构提供了全新的设计范式。
