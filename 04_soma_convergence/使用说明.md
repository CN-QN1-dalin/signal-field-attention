## 📖 简介

Soma Convergence是Soma(Soma)项目的第三大核心组件，采用了革命性的**信号场谐振机制**来替代传统的 KV Cache，实现了：

- **O(1) 内存复杂度**：使用固定 k 个谐振模式替代 KV 序列，内存与序列长度无关
- **O(1) 解码复杂度**：单步解码无需回溯历史，解码延迟恒定
- **真正增量更新**：S_{t+1} = S_t ⊕ x_{t+1}，可随时保存/恢复推理状态

## 🎯 核心特性

| 特性 | 传统 Attention | Soma Convergence |

### 基础用法

```python
import mlx.core as mx
from soma_convergence import SignalFieldIncrementalInference

# 创建信号场增量推理层
dims = 128
num_heads = 4
k = 16  # 谐振模式数量

layer = SignalFieldIncrementalInference(dims, num_heads, k=k)

# 准备输入 [batch, seq, dims]
x = mx.random.normal((1, 32, dims))

# Prefill 阶段：一次性编码并构建推理状态
output, field_state, ring_buffer = layer.prefill(x)

# 单步解码：O(1) 复杂度
x_new = mx.random.normal((1, 1, dims))
output_new, new_field_state, new_ring_buffer = layer.decode_step(
    x_new, field_state, ring_buffer
)

# 继续解码...
```

## 🔬 技术原理

### 信号场表示

Soma Convergence使用 k 个谐振模式来表示历史信息：

```
S = {(A_m, φ_m, ω_m)}_{m=1}^{k}
```

其中：
- A_m = |Σ x_t · e^{-iω_m t}| — 谐振模式的振幅
- φ_m = arg(Σ x_t · e^{-iω_m t}) — 谐振模式的相位
- ω_m = 2πm/k — 谐振模式的频率

### 增量更新

状态更新公式：

```
S_{t+1} = γ · S_t + (1-γ) · k_t
```

其中 γ 是衰减系数，默认值为 0.98。

### 与主流方案对比

| 方案 | 内存复杂度 | 解码复杂度 | 增量更新 | 64K 压缩比 |
|------|-----------|-----------|---------|-----------|
| Attention KV Cache | O(n) | O(n) | ✗ | 1x |
| PagedAttention (vLLM) | O(n) | O(n) | ✗ | ~1x |
| FlashAttention | O(n) | O(n²) | ✗ | 1x |
| Sliding Window | O(w) | O(w) | ✗ | n/w |
| Mamba SSM | O(1) | O(1) | ✗ | - |
| **Soma Convergence** | **O(1)** | **O(1)** | **✓** | **248x** |

## 🔧 API 参考

### SignalFieldIncrementalInference

```python
class SignalFieldIncrementalInference:
    def __init__(self, dims, num_heads, k=16, gamma=0.98, alpha=0.1)
    
    def prefill(self, x) -> Tuple[output, field_state, ring_buffer]
        """Prefill 阶段，返回推理状态"""
    
    def decode_step(self, x_new, field_state, ring_buffer) -> Tuple[output, new_field_state, new_ring_buffer]
        """单步解码，O(1) 复杂度"""
    
    def full_forward(self, x) -> output
        """全量前向传播（参考实现）"""
```

## ⚠️ 局限性

1. **小模型场景**：由于固定的 Ring Buffer 开销，小模型可能略慢于标准 Attention
2. **精度权衡**：使用有限谐振模式可能在某些长距离依赖任务上有精度损失
3. **硬件依赖**：当前仅支持 Apple Silicon (MLX)
