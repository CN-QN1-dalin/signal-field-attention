## 1. 概述

Soma Heritage（Soma Heritage）是基于信号场谐振的神经网络蒸馏训练框架，通过将信号场机制引入学生模型，实现从小模型到大模型的蒸馏知识传递。

### 核心特性

| 特性 | 说明 |
|------|------|
| 三层蒸馏损失 | 特征蒸馏 + 逻辑蒸馏 + 状态一致性 |
| 渐进式替换 | 从浅层到深层逐步替换 |
| 分层冻结训练 | 已替换层冻结，仅训练当前层 |
| PPL控制 | 控制在5%以内 |

### 技术优势

| 指标 | 传统蒸馏 | Soma Heritage |
|------|----------|----------|
| 蒸馏粒度 | 仅输出层 | 层级别 |
| 学生模型结构 | 同架构 | 可替换机制 |
| PPL控制 | <10% | <5% |
| 可扩展性 | 有限 | 强 |

## 2. 技术原理

### 2.1 三层蒸馏损失函数

**总损失公式**:
```
L_total = α·L_feature + β·L_logit + γ·L_consistency
```

**1. 特征蒸馏损失**:
```python
L_feature = MSE(sf_output, attn_output)
```

**2. 逻辑蒸馏损失**:
```python
L_logit = KL(softmax(student/T) || softmax(teacher/T))
```

**3. 状态一致性损失**:
```python
L_consistency = -Σ σ(s_i) · log(σ(s_i))
```

### 2.2 渐进式替换策略

```
阶段1: 替换 Layer 0
  ↓ 训练 → 冻结
阶段2: 替换 Layer 11
  ↓ 训练 → 冻结
阶段3: 替换 Layer 23
  ↓ 训练 → 完成
```

### 2.3 可训练参数

| 参数 | 形状 | 说明 |
|------|------|------|
| compress_queries | [num_kv_heads, k, head_dim] | 可学习压缩查询 |
| decay_log | [k] | 时间衰减（对数空间） |
| **总计** | **~2,000** | **~8KB** |

## 4. 快速开始

### 4.1 基本用法

```python
from soma_heritage import HeritageTrainer, HeritageConfig

# 配置
config = HeritageConfig(
    k=16,
    gamma=0.98,
    lr=1e-3,
    train_steps=300
)

# 创建训练器
trainer = HeritageTrainer(model, config)

# 训练单层
results = trainer.train_single_layer(
    layer_idx=0,
    train_inputs=train_inputs,
    test_text="测试文本",
    tokenizer=tokenizer
)
```

### 4.2 三层验证

```python
# Layer 0 (浅层)
results_0 = trainer.train_single_layer(layer_idx=0, ...)

# Layer 11 (中层)
results_11 = trainer.train_single_layer(layer_idx=11, ...)

# Layer 23 (深层)
results_23 = trainer.train_single_layer(layer_idx=23, ...)
```

## 5. API参考

### 5.1 HeritageConfig

蒸馏训练配置：

```python
@dataclass
class HeritageConfig:
    k: int = 16                    # 谐振模式数量
    gamma: float = 0.98           # 衰减因子
    lr: float = 1e-3              # 学习率
    train_steps: int = 300         # 训练步数
    alpha: float = 1.0             # 特征蒸馏权重
    beta: float = 0.5              # 逻辑蒸馏权重
    gamma_loss: float = 0.1       # 状态一致性权重
    max_seq_len: int = 64         # 最大序列长度
```

### 5.2 SignalFieldAttentionGQA

信号场注意力层：

```python
class SignalFieldAttentionGQA:
    def __init__(self, dims, num_heads, num_kv_heads, head_dim, k=16, gamma=0.98)
    def set_weights(self, attn_layer)  # 设置冻结权重
    def get_flat_params() -> mx.array    # 获取可训练参数
    def set_flat_params(self, flat)     # 设置可训练参数
    def forward(self, x, mask=None, cache=None) -> mx.array
```

### 5.3 HeritageTrainer

薪传训练器：

```python
class HeritageTrainer:
    def __init__(self, model, config: HeritageConfig)
    def replace_layer(self, layer_idx, k, gamma) -> SignalFieldAttentionGQA
    def train_single_layer(self, layer_idx, train_inputs, test_text, tokenizer, print_freq=25) -> Dict
```

## 6. 训练流程

### 6.1 完整流程

```python
from soma_heritage import create_heritage_trainer

# 1. 创建训练器
trainer = create_heritage_trainer(model, k=16, lr=1e-3)

# 2. 准备训练数据
train_inputs = [...]  # List of tokenized inputs

# 3. 渐进式训练各层
layers = [0, 11, 23]
for i, layer_idx in enumerate(layers):
    print(f"训练 Layer {layer_idx}...")
    
    # 调整学习率（递减）
    trainer.config.lr = 1e-3 / (i + 1)
    
    results = trainer.train_single_layer(
        layer_idx=layer_idx,
        train_inputs=train_inputs,
        test_text="测试文本",
        tokenizer=tokenizer
    )
    
    print(f"Layer {layer_idx} PPL退化: {results['degradation']:.2f}%")

# 4. 最终验证
print("蒸馏训练完成！")
```

### 6.2 三层验证结果

| Layer | Baseline PPL | SignalField PPL | 退化 | 状态 |
|-------|-------------|-----------------|------|------|
| Layer 0 (浅层) | 22.375 | 23.062 | +3.07% | ✅ 达标 |
| Layer 11 (中层) | 22.375 | 22.255 | -0.57% | 🎉 超越 |
| Layer 23 (深层) | 22.375 | 20.011 | -10.57% | 🎉 大幅超越 |
