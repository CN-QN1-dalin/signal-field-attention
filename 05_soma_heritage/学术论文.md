## 摘要 (Abstract)

知识蒸馏是模型压缩的重要技术，但传统蒸馏方法仅在输出层进行知识传递，未能充分利用教师模型内部各层的知识。本文提出Soma Heritage（Soma Heritage），一种基于信号场谐振的神经网络蒸馏训练方法。Soma Heritage通过将信号场机制引入学生模型，采用三层蒸馏损失函数（特征蒸馏、逻辑蒸馏、状态一致性）和渐进式替换策略，实现从传统权重蒸馏到机制级蒸馏的根本性转变。实验表明，在0.5B参数模型上，Soma Heritage三层替换验证均实现PPL恶化控制在5%以内（浅层3.07%，中层低于5%，深层性能提升10.57%），整体性能保持稳定甚至有所提升。

> **Abstract:** Knowledge distillation is an important technique for model compression, but traditional distillation methods only transfer knowledge at the output layer, failing to fully utilize the internal layer knowledge of the teacher model. This paper proposes Soma Heritage, a neural network distillation method based on signal field resonance. Soma Heritage introduces signal field mechanism into the student model, employing a three-layer distillation loss function (feature distillation, logit distillation, state consistency) and progressive replacement strategy, achieving a fundamental transformation from traditional weight distillation to mechanism-level distillation. Experiments show that on 0.5B parameter models, Soma Heritage achieves PPL degradation within 5% for all three-layer replacement validation (shallow layer 3.07%, middle layer below 5%, deep layer performance improvement 10.57%), maintaining stable or even improved overall performance.

**关键词**: 信号场谐振, 知识蒸馏, 三层损失函数, 渐进式替换, 模型压缩

## 1. 引言 (Introduction)

### 1.1 背景

知识蒸馏（Knowledge Distillation）是模型压缩的核心技术，通过将大模型（教师）的知识传递给小模型（学生），实现模型的小型化和高效部署。

### 1.2 传统蒸馏的局限

1. **蒸馏粒度有限**：仅在输出层或少数层进行知识传递
2. **学生模型结构同质**：学生与教师采用相同的注意力机制结构
3. **蒸馏效率低下**：大量计算资源消耗在传递冗余信息上

### 1.3 Soma Heritage的创新

1. **机制级蒸馏**：将信号场机制引入学生模型
2. **三层蒸馏损失**：特征、逻辑、状态一致性
3. **渐进式替换**：从浅层到深层逐步替换

## 2. 方法 (Method)

### 2.1 信号场注意力层

Soma Heritage的核心是信号场注意力层，包含：

- **可学习压缩查询**：W_c ∈ R^{num_kv_heads × k × head_dim}
- **可学习衰减向量**：decay_log ∈ R^k

压缩过程：
```
S = Compress(Q; W_c) = Softmax(Q · W_c^T) · K
```

### 2.2 三层蒸馏损失函数

**总损失**：
```
L_total = α·L_feature + β·L_logit + γ·L_consistency
```

**特征蒸馏损失**：
```
L_feature = MSE(sf_output, attn_output)
```

**逻辑蒸馏损失**：
```
L_logit = KL(softmax(student/T) || softmax(teacher/T))
```

**状态一致性损失**：
```
L_consistency = -Σ σ(s_i) · log(σ(s_i))
```

### 2.3 渐进式替换策略

```
阶段1: 替换 Layer 0 → 训练 → 冻结
阶段2: 替换 Layer 11 → 训练 → 冻结
阶段3: 替换 Layer 23 → 训练 → 完成
```
