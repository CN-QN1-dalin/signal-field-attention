# 安全政策

## 受支持版本

| 版本 | 支持中 |
|------|--------|
| 0.1.0 | ✅ |
| 0.2.0 | ✅ |
| 1.0.0 | ✅ |

## 报告漏洞

请报告安全漏洞：

1. **不要**在公共 Issue 中报告
2. 发送邮件至 security@dalan-lang.org
3. 包含详细的问题描述
4. 提供复现步骤
5. 注明受影响版本

## 响应时间

- **严重漏洞**: 24 小时内响应
- **一般漏洞**: 72 小时内响应
- **低危漏洞**: 1 周内响应

## 处理流程

1. 确认问题
2. 开发修复
3. 发布补丁
4. 通知用户
