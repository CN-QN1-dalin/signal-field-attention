Dalin Jia
Independent Researcher

Preprint — Preliminary results, simulation-based experiments included

Abstract

Transformer-based large language models suffer from inherent O(n2) computational complexity and O(n) KV cache memory growth during autoregressive decoding, creating fundamental bottlenecks for long-context deployment. We propose Soma, a signal field native architecture that replaces standard self-attention with a dual-channel mechanism combining near-field precise computation (Ring Buffer) and far-field compressed aggregation (Exponential Moving Average state). Our approach achieves O(k ⋅ n) computational complexity and O(k) memory complexity where k is a fixed window size, decoupling both from sequence length n.

On Qwen2.5-7B-Instruct (4-bit), Soma achieves 284×–567× KV cache compression at 64K sequence (462 KB vs 128–256 MB depending on precision), with Cosine Similarity  > 0.9999999 to standard causal attention for t ≥ 1 (verified across 7 sequence lengths). The framework adds only ∼8.1–32.8 KB of trainable parameters. Single-token decoding latency is O(1) constant (∼0.52 ms/step, coefficient of variation 0.63%) in our MLX prototype.

We further propose three extensions: (1) Soma Heritage, a distillation framework using signal field attention in student models with progressive layer replacement; (2) Soma Native, a full architecture replacing all Transformer components (attention, FFN, LayerNorm, RoPE) with signal field equivalents; (3) Soma LingYa, a parameter-efficient fine-tuning method achieving 50% parameter reduction versus LoRA at equivalent rank.

All experiments reported herein are from MLX Python prototype implementations. Simulated data is explicitly labeled. C++/Metal kernel deployment targets are noted separately from verified results.

Keywords: Long-context LLM, KV cache compression, signal field attention, parameter-efficient fine-tuning, knowledge distillation, O(1) memory

Introduction

The Transformer architecture has dominated natural language processing since 2017. Its core innovation—self-attention—enables global token interaction but introduces two fundamental bottlenecks for long-sequence processing:

Computational complexity: Standard self-attention requires O(n2 ⋅ d) operations for sequence length n and model dimension d. During autoregressive decoding, each new token must attend to all previous tokens, causing quadratic slowdown.

Memory complexity: The KV cache grows linearly as O(n ⋅ d), reaching hundreds of megabytes for 7B models at 64K sequence. This limits practical context length and increases deployment costs.

Existing solutions fall into several categories, each with trade-offs:

  • Sparse/sliding window attention (StreamingLLM , H2O , SnapKV ): Discard distant tokens, losing long-range dependencies.

  • Linear attention (LinFormer , Performer ): Approximate softmax attention via low-rank projection or random feature maps, introducing approximation error.

  • State Space Models (Mamba , RWKV ): Achieve O(n) complexity but cannot inherit pre-trained Transformer weights and require full retraining.

  • KV cache compression (quantization, pooling, low-rank): Reduce memory but introduce irreversible precision loss.

  • FlashAttention : Optimizes I/O patterns but does not change asymptotic complexity.

We propose Soma, a signal field native architecture that addresses the accuracy-efficiency trade-off without breaking Transformer compatibility. Our core insight: treat discrete token sequences as continuous temporal signals. Recent tokens are preserved precisely for detailed reasoning, while distant history is compressed into fixed-dimensional field states via exponential moving average (EMA) decay.

  2. We demonstrate 284×–567× KV cache compression at 7B/64K with only 8.1–32.8 KB parameter overhead.

  3. We propose Soma Heritage, a distillation framework with progressive layer replacement and adaptive loss weighting.

  4. We propose Soma Native, a full architecture replacing all Transformer components.

  5. We propose Soma LingYa, a PEFT method with 50% parameter reduction versus LoRA.

Signal Field Attention

Dual-Channel Mechanism

Given query qt, key sequence K = {k1, …, kn}, value sequence V = {v1, …, vn} at decoding step t, SFA computes:

Attention(qt, K, V) = Attentionnear(qt, Kring, Vring) + α ⋅ Attentionfar(qt, SK, SV)

Near-field channel: Standard softmax attention on the most recent k tokens stored in a Ring Buffer:

$$\begin{equation} \text{Attention}_{\text{near}} = \text{softmax}\left(\frac{q_t \cdot K_{\text{ring}}^\top}{\sqrt{d_k}}\right) \cdot V_{\text{ring}} \end{equation}$$

Far-field channel: Attention against EMA-compressed field states:

SK(t) = γ ⋅ SK(t − 1) + (1 − γ) ⋅ kt,  SV(t) = γ ⋅ SV(t − 1) + (1 − γ) ⋅ vt

$$\begin{equation} \text{Attention}_{\text{far}} = \frac{q_t \cdot S_K^{(t)}}{\sqrt{d_k}} \cdot S_V^{(t)} \end{equation}$$

where γ ∈ (0, 1) is the temporal decay coefficient (default 0.98), and α is the far-field mixing coefficient (default 0.1).

Complexity Analysis

Theorem 1 (Complexity Bounds). For sequence length n, model dimension d, window size k, and number of attention heads h:

$$\begin{align} T_{\text{compute}} &= O(n \cdot (k \cdot d + d)) = O(k \cdot n \cdot d) \\ M_{\text{memory}} &= O(k \cdot h \cdot d_{\text{head}} + h \cdot d_{\text{head}}) = O(k \cdot d) \end{align}$$

Proof: The near-field channel processes k tokens per step, requiring O(k ⋅ d) per token. The far-field channel operates on fixed-size state vectors of dimension dhead, requiring O(d) per step. Summing over n tokens yields O(k ⋅ n ⋅ d) computation and O(k ⋅ d) memory. ▫

Corollary 1. When k ≪ n, computation approaches linear O(n ⋅ d) while memory is strictly constant O(k ⋅ d), independent of sequence length.

Soma Heritage: Distillation Framework

Progressive Layer Replacement

Instead of replacing all attention layers simultaneously, Soma Heritage progressively replaces layers from shallow to deep, freezing each replaced layer before proceeding to the next. This prevents covariate shift from simultaneous input distribution changes.

Input: Teacher MT, Student MS, replacement schedule {L1, …, Lm} l ← Lstage Replace MS.layers[l].attention with SignalFieldAttention Freeze MS.layers[i] Initialize trainable SFA parameters for layer l Compute three-layer distillation loss (Sec. 4.2) Update trainable parameters of layer l only Freeze layer l

Three-Layer Distillation Loss

ℒtotal = w1 ⋅ ℒfeature + w2 ⋅ ℒlogit + w3 ⋅ ℒconsistency

where ℒfeature constrains SFA layer output vs teacher attention output, ℒlogit matches logit distributions via KL divergence, and ℒconsistency constrains SFA internal state stability.

Weights are adjusted via GradNorm to balance gradient magnitudes across losses.

Homeostasis and GrowthTemporal

Homeostasis replaces LayerNorm with dynamic per-channel scaling factors that adapt via exponential moving average, rather than using fixed batch statistics.

GrowthTemporal replaces static RoPE with learnable basis functions for positional encoding, allowing the model to adapt its temporal representation during training.

Note: These components have theoretical definitions but no experimental validation yet. Their effectiveness must be verified in full model training.

Soma LingYa: Parameter-Efficient Fine-Tuning

Method

Soma LingYa replaces LoRA’s low-rank decomposition ΔW = B ⋅ A with a gated modulation framework:

ΔW = R ⋅ P ⋅ α

where R ∈ ℝdout × r is a frozen scaffold matrix, P ∈ ℝr × din is a zero-initialized growth matrix trained from scratch, and α is a growth scale factor.

Theorem 2 (Parameter Efficiency). At equivalent rank r, LingYa requires dr trainable parameters versus LoRA’s 2dr, achieving 50% parameter reduction.

Proof: LoRA trains both B (dout × r) and A (r × din), totaling 2dr parameters when dout = din = d. LingYa trains only P (r × din), totaling dr parameters. Ratio: dr/2dr = 1/2. ▫

Scaffold Matrix Design

Scaffold matrix initialization strategies.
Channel Type Initialization Property
ROOT R = I[:, : r] Orthogonal projection
BRANCH R = Ur (SVD of random matrix) Orthogonal basis
LEAF R = ϵ ⋅ Z, ϵ ∼ 𝒩(0, 0.02) Small perturbation

Delta Clamp

To prevent P from diverging during training, we constrain its Frobenius norm:

$$\begin{equation} \text{if } \|P\|_F > \tau_{\text{max}}: \quad P \leftarrow P \cdot \frac{\tau_{\text{max}}}{\|P\|_F} \end{equation}$$

with τmax = 5.0 (recommended).

Limitations

  1. Prototype only: All data from MLX Python prototype. C++/Metal deployment targets (4.16× speedup) are not yet verified.

  2. Simulated distillation data: PPL values, ablation studies, and downstream task results for Heritage are formula-generated. Real distillation training is required.

  3. Unvalidated components: Homeostasis, GrowthTemporal, and channel ablation studies lack experimental verification.

  4. Single model size: Primary experiments on 0.5B model. 7B+ verification needed.

  5. Missing SOTA comparisons: No comparison with QLoRA, DoRA, AdaLoRA (PEFT), or Mamba-2, RWKV-6 (sequence models).

  6. Single seed: All experiments use seed=42. Multiple seeds needed for statistical significance.

Theoretical Limitations

Preliminary theoretical analyses (information capacity bounds, distance kernel approximation) contain known issues: inequality direction errors and unjustified assumptions about the relationship between attention matrix condition number and EMA decay. These provide intuition but are not rigorous proofs. Formal theoretical analysis is future work.

Conclusion

We present Soma, a signal field native architecture for efficient long-sequence processing. Key verified results:

  • Correctness: Cosine Similarity  = 1.000000 to causal standard attention for t ≥ 1 (7 sequence lengths)

  • Memory: 284×–567× compression at 7B/64K (462 KB vs 128–256 MB)

  • Decode: O(1) constant latency (CV = 0.63%)

  • Parameters: 8.1–32.8 KB overhead

  • PEFT: 50% parameter reduction vs LoRA (proven)

Simulated results (Heritage PPL, channel ablation, latency estimates) are clearly labeled and represent research directions requiring future experimental validation.
