## 1. 概述

Soma Native（Soma Native Architecture）是从零设计的神经网络架构，完全基于信号场机制，替代Transformer的全部核心组件。

### 架构对比

| 组件 | Transformer | Soma Native |
|------|-------------|-------------|
| 信息交互 | 多头自注意力 O(n²) | 信号场层 O(k·n) |
| 知识存储 | 前馈网络层 | 灵芽块 |
| 归一化 | LayerNorm | 稳态调节 |
| 位置编码 | RoPE/绝对位置 | 生长时序 |
| 整体复杂度 | O(n²) | O(k·n) |

### 核心优势

1. **架构统一简洁**：一个Soma块替代原来两种异构组件
2. **计算效率高**：O(k·n)复杂度 vs O(n²)
3. **内存效率高**：KV缓存O(k·d) vs O(n·d)
4. **参数效率高**：约8KB替代整个注意力机制
5. **原生设计**：不受Transformer历史包袱限制

## 2. 架构设计原理

### 2.1 信号场理论

Soma架构的核心是信号场（Signal Field），源自神经科学的场电位理论：

- 每个神经元向外广播信号
- 信号强度随距离衰减（高斯衰减）
- 多个信号相遇时产生场干涉

### 2.2 统一场块

```
输入 x
    │
    ▼
生长时序 TE ──► 时序调制信号
    │
    ▼
信号场 S ◄── 邻域神经元激活
    │
    ├──► 场共振 R
    │
    ▼
灵芽通道 W ◄── R + TE
    │
    ▼
稳态调节 H ◄── 活跃度监控
    │
    ▼
输出 o
```

## 3. 核心组件

### 3.1 SignalFieldLayer（信号场层）

替代多头自注意力机制：
- 双通道注意力：近场Ring Buffer + 远场Field State
- O(k·n)复杂度
- 可学习的压缩查询

### 3.2 LingYaBlock（灵芽块）

替代前馈网络层：
- 门控调制机制
- 灵芽P矩阵生长
- 参数量远小于FFN

### 3.3 Homeostasis（稳态调节）

替代LayerNorm：
- 动态平衡而非归一化
- 监控神经元活跃度
- 自动调节

### 3.4 GrowthTemporal（生长时序）

替代位置编码：
- 时序信息内嵌于结构
- 可学习的时间戳
- 无需外部位置编码

## 4. 快速开始

### 4.1 创建模型

```python
from soma_native import create_soma_brain

# 创建小模型
model = create_soma_brain('small', vocab_size=10000)

# 自定义配置
from soma_native import SomaBrain, SomaConfig

config = SomaConfig(
    vocab_size=50000,
    d_model=512,
    num_layers=12,
    num_heads=8,
    max_seq_len=2048
)
model = SomaBrain(config)
```

### 4.2 前向传播

```python
import mlx.core as mx

# 输入
batch_size = 2
seq_len = 32
input_ids = mx.random.randint(0, 10000, (batch_size, seq_len))

# 前向传播
logits = model(input_ids)
print(f"输出形状: {logits.shape}")  # (2, 32, 10000)
```

## 5. API参考

### 5.1 SomaConfig

配置数据类：

```python
@dataclass
class SomaConfig:
    vocab_size: int = 50000
    d_model: int = 512
    num_layers: int = 8
    num_heads: int = 8
    max_seq_len: int = 2048
    k: int = 16  # 信号场谐振模式数量
```

### 5.2 SomaBrain

主模型类：

```python
class SomaBrain:
    def __init__(self, config: SomaConfig)
    def __call__(self, input_ids: mx.array) -> mx.array
    def get_num_params(self) -> int
```

### 5.3 工厂函数

```python
create_soma_brain(size='small', vocab_size=50000)
```

可选size: 'small', 'medium', 'large'

## 6. 配置参数

### 6.1 预置配置

| 配置 | 维度 | 层数 | 头数 | 适用场景 |
|------|------|------|------|----------|
| small | 256 | 6 | 4 | 快速实验 |
| medium | 512 | 12 | 8 | 标准训练 |
| large | 768 | 16 | 12 | 大规模部署 |

### 6.2 自定义参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| vocab_size | 50000 | 词汇表大小 |
| d_model | 512 | 模型隐藏维度 |
| num_layers | 8 | Soma块数量 |
| num_heads | 8 | 注意力头数量 |
| max_seq_len | 2048 | 最大序列长度 |
| k | 16 | 信号场谐振模式数量 |
