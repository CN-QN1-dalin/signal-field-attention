# Soma Engine - 实验成功代码包

### 核心实验结果

- **测试环境**：Apple M1 Pro, 32GB RAM
- **测试模型**：Qwen-7B / Qwen-14B
- **层替换比例**：71%（第1-6, 9-14, 17-22层，共15层）
- **端到端加速**：+19%（平均），+34%（32K长序列）
- **PPL精度损失**：0.9%
- **内存增量**：0%
