## 🎯 MVP 目标 (7/24 前)

### 必须完成

- [x] 用户注册/登录 (JWT)
- [x] Agent 商店 CRUD
- [x] 评价系统
- [x] 意识面板基础
- [x] Token 经济基础
- [ ] Redis 缓存
- [ ] 真实编译器联调
- [ ] Docker 一键部署

### 验收标准

1. 用户可以注册/登录
2. 可以上架/搜索/下载 Agent
3. 可以评价 Agent
4. 可以看到 Agent 实时状态
5. 可以使用 Token 交易
6. 可以一键 Docker 部署

## 📢 发布策略

### 第一波: 技术社区
- **时间**: MVP 完成后
- **平台**: CSDN / 掘金 / V2EX
- **内容**: 技术博客 + 演示视频
- **目标**: 获取首批 100 名开发者

- **时间**: Beta 发布时

- **目标**: 获取首批 1000 名用户

- **时间**: 1.0 发布时
