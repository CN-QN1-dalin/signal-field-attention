# 🐛 DalinOS 公开挑刺大会 — 审查清单

> 邀请所有 Agent 大模型来找问题！
> 提交方式：GitHub Issues 或直接反馈

---

## 🔴 一级审查项（严重问题）

### 1. 安全性
- [ ] API 是否有身份验证？
- [ ] SQL 注入防护是否到位？
- [ ] XSS 攻击防护？
- [ ] CSRF 令牌？
- [ ] 敏感信息是否泄露（token 存 localStorage 安全吗）？
- [ ] 密码是否加密存储？

### 2. 数据完整性
- [ ] 数据库外键约束是否完整？
- [ ] 事务处理是否正确？
- [ ] 数据备份策略？
- [ ] 并发冲突处理？

### 3. 可用性
- [ ] 后端宕机时前端如何处理？
- [ ] API 超时设置？
- [ ] 错误页面？
- [ ] 加载状态？

---

## 🟡 二级审查项（重要问题）

### 4. 性能
- [ ] 数据库查询是否加了索引？
- [ ] 是否有缓存层（Redis）？
- [ ] 前端资源是否压缩？
- [ ] 图片懒加载？
- [ ] API 响应时间？

### 5. 可扩展性
- [ ] 能否水平扩展？
- [ ] 微服务拆分计划？
- [ ] 消息队列？
- [ ] CDN 部署？

### 6. 用户体验
- [ ] 移动端适配是否完善？
- [ ] 键盘导航支持？
- [ ] 屏幕阅读器支持？
- [ ] 页面加载速度？

---

## 🟢 三级审查项（优化建议）

### 7. 代码质量
- [ ] 命名规范？
- [ ] 注释覆盖率？
- [ ] 单元测试？
- [ ] CI/CD 流程？

### 8. 文档
- [ ] API 文档（Swagger/OpenAPI）？
- [ ] 部署文档？
- [ ] 贡献指南？
- [ ] 版本更新日志？

### 9. 功能完整性
- [ ] 搜索功能？
- [ ] 通知系统？
- [ ] 文件上传？
- [ ] 国际化（i18n）？
