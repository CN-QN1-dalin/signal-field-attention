## 1. 项目概述

**Dalin ISFE (Intent Signal Field Engine)** 是全球首个将 SFA 信号场思想应用到意图理解的引擎。

**核心创新**:
- 把 SFA 的三通道机制（RingBuffer/EMA/Semantic Pool）应用到意图理解
- 让 AI 从"被动对齐"升级为"主动理解"
- 原创性技术，没有直接竞争对手

## 3. 数据流

```
自然语言输入
    ↓
嵌入模型编码 (128 维向量)
    ↓
┌─────────────────────────────────────────┐
│  并行处理三通道:                         │
│                                         │
│  RingBuffer: 添加到缓冲区，计算均值       │
│  EMA Field: EMA[t] = γ·EMA[t-1] + (1-γ)·intent│
│  Semantic Pool: Attention 加权查询       │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  融合: enhancement = ring_mean + 0.5·ema│
│                    + 0.5·semantic        │
└─────────────────────────────────────────┘
    ↓
意图理解结果 (向量)
    ↓
解码器映射回自然语言
    ↓
输出: {intent: "xxx", confidence: 0.99}
```

## 4. 核心数学

### 4.1 意图嵌入

```
intent_embedding = Embedder(natural_language)

Embedder: 预训练嵌入模型 (128 维)
natural_language: "根据病情决定是否需要住院"
intent_embedding: [0.12, -0.34, 0.56, ..., 0.23] (128 维)
```

### 4.2 RingBuffer 均值

```
ring_mean = (1/N) · Σᵢ intent[i]

N: 缓冲区大小 (16)
intent[i]: 第 i 轮意图向量
```

### 4.3 EMA Field

```
EMA[t] = γ · EMA[t-1] + (1-γ) · intent[t]

γ: 衰减因子 (0.98)
intent[t]: 当前轮意图向量
EMA[t]: 当前轮长期意图趋势
```

### 4.4 Semantic Pool

```
weights = Softmax(intent · K^T / temperature)

K: 语义槽位矩阵 (64 × 128)
temperature: 温度参数 (0.07)
weights: 注意力权重 (64 维)

semantic = W^T · weights

W: 值矩阵 (64 × 128)
semantic: 查询结果 (128 维)
```

### 4.5 融合

```
enhancement = ring_mean + 0.5 · EMA + 0.5 · semantic

ring_mean: RingBuffer 均值 (128 维)
EMA: EMA Field 当前值 (128 维)
semantic: Semantic Pool 查询结果 (128 维)
enhancement: 融合后意图 (128 维)
```

## 5. 接口设计

### 5.1 Python API

```python
from dalin_isfe import IntentSignalFieldEngine

# 初始化引擎
engine = IntentSignalFieldEngine(
    dim=128,
    ring_capacity=16,
    ema_gamma=0.98,
    semantic_slots=64,
    temperature=0.07
)

# 处理对话
result = engine.process_dialogue(
    user_input="根据病情决定是否需要住院",
    ai_response="建议住院观察",
    history=[
        {"user": "患者有高血压", "ai": "建议就医"},
        {"user": "还有糖尿病", "ai": "建议住院"},
    ]
)

print(result)
# {
#     "intent": "medical_diagnosis",
#     "confidence": 0.99,
#     "enhancement": [...],  # 128 维融合意图
#     "ring_mean": [...],    # 128 维 RingBuffer 均值
#     "ema": [...],          # 128 维 EMA 值
#     "semantic": [...]      # 128 维 Semantic Pool 查询
# }

# 获取意图趋势
trend = engine.get_intent_trend()
print(trend)
# {"direction": "increasing_severity", "score": 0.85}
```

### 5.2 Rust API

```rust
use dalin_isfe::IntentSignalFieldEngine;

fn main() {
    // 初始化引擎
    let mut engine = IntentSignalFieldEngine::new(128, 16, 0.98, 64, 0.07);
    
    // 处理对话
    let result = engine.process_dialogue(
        "根据病情决定是否需要住院",
        "建议住院观察",
        &[
            ("患者有高血压", "建议就医"),
            ("还有糖尿病", "建议住院"),
        ],
    );
    
    println!("Intent: {}", result.intent);
    println!("Confidence: {}", result.confidence);
    println!("Enhancement: {:?}", result.enhancement);
}
```
