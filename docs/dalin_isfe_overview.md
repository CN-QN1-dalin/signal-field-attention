## 1. 项目概述

**Dalin ISFE (Intent Signal Field Engine)** 是全球首个将 SFA 信号场思想应用到意图理解的引擎。

**核心创新**:
- 把 SFA 的三通道机制（RingBuffer/EMA/Semantic Pool）应用到意图理解
- 让 AI 从"被动对齐"升级为"主动理解"
- 原创性技术，没有直接竞争对手

**技术栈**:
- **意图 RingBuffer**: 短期意图记忆（16 个槽位）
- **意图 EMA Field**: 长期意图趋势（γ=0.98）
- **意图 Semantic Pool**: 全局意图语义（64 个槽位）
- **意图融合**: 三通道加权融合

## 4. 核心数学

### 4.1 意图 RingBuffer

```
RingBuffer[t] = {intent[t-15], intent[t-14], ..., intent[t]}

大小: 16 个意图槽位
更新: 滑动窗口，每次对话更新最新槽位
```

### 4.2 意图 EMA Field

```
EMA[t] = γ · EMA[t-1] + (1-γ) · intent[t]

γ: 衰减因子 (0.98)
含义: 长期意图趋势，捕捉意图演变
```

### 4.3 意图 Semantic Pool

```
SemanticPool = Attention(intent_embeddings, temperature=0.07)

大小: 64 个语义槽位
含义: 全局意图语义，捕捉深层含义
```

### 4.4 意图融合

```
enhancement = ring_mean + 0.5 · EMA + 0.5 · SemanticPool

ring_mean: RingBuffer 的平均意图
EMA: 长期意图趋势
SemanticPool: 全局意图语义
```
