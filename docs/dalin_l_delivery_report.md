## 三、技术亮点

### 1. 零学习成本
- 25 个关键字（核心 15 + 扩展 10）
- 类型自动推断
- 中文变量名/函数名/注释

### 2. 零碎片化工具链
- `dalin build` — 编译 + 格式化 + 类型检查
- `dalin run` — 运行
- `dalin test` — 测试
- `dalin repl` — 交互式开发
- `dalin fmt` — 格式化
- `dalin docs` — 文档生成

### 3. 零不确定性
- HM 类型推断（理论保证）
- 编译期确定（无运行时意外）
- 中文错误信息（清晰易懂）

### 4. Agent 原生设计
- 上下文感知编译
- 自我修复编译（概率性 + 确认）
- 自动测试生成
- 统一 FFI
- 代码即对话

### 5. 性能碾压
- 编译像 Go 一样快
- 运行像 C 一样快
- 内存像 C 一样少
- 二进制像 C 一样小
