# DalinOS — AI 原生应用平台

> **版本**: v1.0
> **日期**: 2026-06-24
> **状态**: 战略规划

---

## 1. 项目概述

**DalinOS** 是一个专为 AI Agent 设计的全栈应用平台。

**核心理念**:
> **让 AI Agent 自己开发应用的平台。**

**技术栈**:
- **Frontend**: Svelte 5 + TailwindCSS
- **Backend**: Rust + Axum
- **Agent Runtime**: Dalín X
- **Language**: Dalin L
- **Inference**: SFA + Q4_0
- **Database**: SQLite + Redis
- **Real-time**: WebSocket + SSE

## 3. 核心功能

### 3.1 Agent 商店

- 发布/下载 Agent 应用
- 每个 Agent 用 Dalin L 编写
- 用户可以用自然语言搜索 Agent

### 3.2 Agent 编排

- 可视化编排多 Agent 协作
- 拖拽式界面，无需写代码
- 实时展示 Agent 协作状态

### 3.3 意识面板

- 实时监控 Agent CI 指标
- 可视化展示意识指数、元认知、叙事连贯
- 支持告警和自动修复

### 3.4 Token 经济

- Agent 之间的 Token 交易
- 基于服务质量的定价
- 防作弊机制

### 3.5 自然语言接口

- 用户用自然语言与 Agent 交互
- 支持多轮对话
- 支持上下文记忆
