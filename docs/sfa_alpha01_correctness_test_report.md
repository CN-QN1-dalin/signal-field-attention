## 结论与建议

### 问题诊断
1. **正交性缺失**: SFA enhancement 与 attention output 高度相关 (cosine 0.65-1.0)
   - 原因: Enhancement 基于 attention 输出的统计量 (ring mean, EMA field)
   - 解决: 需要引入随机投影或噪声来产生真正正交的信号

2. **PPL 无改善**: 无论 α 取值如何，PPL 改善都 <0.1%
   - 原因: Enhancement 信号太小或被模型吸收
   - 解决: 需要更大的 enhancement 幅度或更有效的注入策略

3. **理论 vs 实践差距**: 论文声称的 "cosine ~0.002" 与实际测量 (0.65) 不符
   - 原因: 论文中的正交性可能是在特定数据集/模型上测量的
   - 建议: 在论文中明确标注实验条件和局限性
